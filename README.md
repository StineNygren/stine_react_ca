## Description
A online shop were you can

- Search trough products
- Add product to cart
- Edit products in cart
- "BUY" products
- Submit contact form to console

## Built With

- Type script
- React
- Redux
- MUI

## Getting Started

1. Clone the repo:

```bash
git clone https://github.com/StineNygren/stine_react_ca.git
```

2. Install the dependencies:

```
npm install
```
